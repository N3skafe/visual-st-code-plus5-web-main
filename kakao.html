배열을 활용한 카카오 코딩 테스트 문제 풀이

def solution(record):
    # 사용자 ID와 닉네임을 매핑할 딕셔너리
    user_dict = {}
    # 각 행동(Enter, Leave, Change)과 사용자 ID를 저장할 배열
    actions = []
    
    # 모든 기록을 순회하며 사용자 정보와 행동을 저장
    for r in record:
        parts = r.split()
        action = parts[0]
        user_id = parts[1]
        
        # Enter와 Change 액션일 경우 닉네임 업데이트
        if action in ["Enter", "Change"]:
            nickname = parts[2]
            user_dict[user_id] = nickname
        
        # Enter와 Leave 액션만 메시지 출력 대상
        if action != "Change":
            actions.append((action, user_id))
    
    # 결과 메시지 생성
    result = []
    for action, user_id in actions:
        if action == "Enter":
            result.append(f"{user_dict[user_id]}님이 들어왔습니다.")
        elif action == "Leave":
            result.append(f"{user_dict[user_id]}님이 나갔습니다.")
    
    return result

📚 마법의 채팅방 이야기 📚

옛날 옛적에, 아이들이 모여 놀 수 있는 신기한 마법의 놀이방이 있었어요. 🏰✨ 이 놀이방에는 많은 친구들이 들어오고 나가면서 재미있게 놀았답니다.

친구들은 놀이방에 들어올 때마다 자신의 비밀 이름표를 가지고 들어왔어요. 🏷️ 그리고 자기가 불리고 싶은 별명을 정해서 친구들에게 알려주곤 했죠. 때로는 놀다가 갑자기 "나 이제 공룡🦖이 아니라 우주인👨‍🚀으로 불러줘!"라고 별명을 바꾸기도 했어요.

마법의 놀이방 문 앞에는 신기한 요정이 있었는데요, 이 요정은 친구들이 들어오고 나갈 때마다 기록을 남겼어요. 📝✨ "○○이가 들어왔어요!" 또는 "△△이가 나갔어요!"라고요.

하지만 요정에게는 큰 고민이 있었어요. 친구들이 별명을 자꾸 바꾸면, 마지막에 누가 어떤 별명이었는지 기억하기 어려웠거든요. 😵‍💫 요정은 모든 친구를 항상 최신 별명으로 불러주고 싶었답니다.

그래서 요정은 두 가지 마법 주머니를 만들었어요:

  • 첫 번째 주머니👝에는 친구들의 비밀 이름표와 최신 별명을 함께 넣어두었어요.
  • 두 번째 주머니👜에는 친구들이 들어오고 나간 순서대로 행동을 기록했어요.

놀이가 모두 끝나고 나서, 요정은 첫 번째 주머니👝에서 각 친구의 최신 별명을 확인하고, 두 번째 주머니👜에 있는 기록을 보면서 "○○이가 들어왔어요!" 또는 "△△이가 나갔어요!"라고 정확하게 말할 수 있게 되었답니다! 🎉

이렇게 요정은 친구들의 별명이 바뀌어도 항상 최신 별명으로 기록을 남길 수 있게 되었어요. 그리고 모든 아이들은 자신의 원하는 별명으로 불리며 즐겁게 놀 수 있었답니다. 🌈👭👬

원형 링크드 리스트 - 카카오 코딩 테스트 문제 풀이

원형 링크드 리스트를 활용한 카카오 코딩 테스트 문제 풀이

class Node:
    def __init__(self, value=None):
        self.value = value
        self.next = None

def solution(n, k):
    # 요세푸스 문제: n명이 원형으로 앉아있고, k번째 사람을 계속 제거할 때 제거되는 순서 구하기
    
    # 원형 링크드 리스트 생성
    if n <= 0 or k <= 0:
        return []
    
    # 첫 번째 노드 생성
    head = Node(1)
    current = head
    
    # 2부터 n까지의 노드 생성 및 연결
    for i in range(2, n + 1):
        current.next = Node(i)
        current = current.next
    
    # 원형 링크드 리스트로 만들기 위해 마지막 노드를 첫 번째 노드에 연결
    current.next = head
    
    # 제거 순서 기록 배열
    result = []
    
    # n-1번 제거 (마지막 한 명은 따로 처리)
    prev = current  # 마지막 노드를 이전 노드로 설정
    current = head  # 현재 위치를 첫 번째 노드로 설정
    
    while current.next != current:  # 노드가 하나만 남을 때까지
        # k-1번 이동
        for _ in range(k - 1):
            prev = current
            current = current.next
        
        # k번째 노드 제거
        result.append(current.value)
        prev.next = current.next
        current = current.next
    
    # 마지막 남은 노드 추가
    result.append(current.value)
    
    return result

🎪 둥글게 둥글게 사라지는 친구들 🎪

옛날 옛적에, 동그란 탁자 🔄 주변에 앉아있는 친구들이 있었어요. 모두 n명의 친구들이 번호를 1부터 차례로 받아서 앉았답니다. 👧👦👧👦

친구들은 "둥글게 둥글게 술래잡기" 🎮 게임을 하기로 했어요. 게임의 규칙은 이랬어요:

1️⃣ 1번 친구부터 시작해서 시계방향으로 숫자를 세어요.
2️⃣ 숫자를 k번째까지 세면, 그 자리에 앉은 친구가 게임에서 나가요. 👋
3️⃣ 그리고 나간 친구의 다음 친구부터 다시 1부터 숫자를 세기 시작해요.
4️⃣ 이렇게 한 명만 남을 때까지 계속해요.

예를 들어, 7명의 친구가 있고 3번째마다 친구를 내보낸다면:

  • 1, 2, (3) - 3번 친구가 나갔어요! 🚶
  • 4, 5, (6) - 6번 친구가 나갔어요! 🚶
  • 7, 1, (2) - 2번 친구가 나갔어요! 🚶
  • 4, 5, (7) - 7번 친구가 나갔어요! 🚶
  • 1, (4) - 4번 친구가 나갔어요! 🚶
  • 5, (1) - 1번 친구가 나갔어요! 🚶

그리고 마지막에는 5번 친구만 남았어요! 🎉

이렇게 친구들이 나가는 순서는 [3, 6, 2, 7, 4, 1, 5] 였답니다!

이 게임은 마치 친구들이 손을 잡고 ⭕ 원을 그리다가 한 명씩 손을 놓고 빠져나가는 것 같아요. 동그란 기차가 한 칸씩 사라지는 것처럼요! 🚂✨

컴퓨터 친구 👨‍💻는 이 게임을 기억하기 위해 특별한 방법을 사용했어요. 각 친구를 작은 상자📦로 만들고, 각 상자는 다음 친구를 가리키는 화살표➡️를 가지고 있었죠. 마지막 친구의 화살표는 다시 첫 번째 친구를 가리키게 해서 동그란 원 ⭕이 되었답니다!
